# DQODE - Diagram Generation Tool

This project generates People, Technology, and Process diagrams from an Excel input file.

--------------------------------------------------
📌 INPUT FILE
--------------------------------------------------

Place your input Excel file in the project directory:

sample_questions.xlsm

(Sheet name should be: Q_Stories)

--------------------------------------------------
⚙️ SETUP (FIRST TIME ONLY)
--------------------------------------------------

1. Create virtual environment:

    python -m venv venv

2. Activate virtual environment:

    Windows:
    venv\Scripts\activate

3. Install dependencies:

    pip install -r requirements.txt

--------------------------------------------------
▶️ EXECUTION
--------------------------------------------------

Run the main program:

    python main.py

This will generate all diagrams:
- People Diagram
- Technology Diagram
- Process Network Diagram

--------------------------------------------------
🎯 OPTIONAL: RUN SPECIFIC DIAGRAM

You can run specific diagram types using arguments:

    python main.py people
    python main.py technology
    python main.py process
    python main.py all

--------------------------------------------------
📤 OUTPUT FILES
--------------------------------------------------

The following files will be generated:

- Diagram_People_New
- Diagram_Technology_New
- Diagram_Network_New

These are Graphviz DOT format files.

--------------------------------------------------
📊 VISUALIZATION
--------------------------------------------------

To view diagrams:

1. Open:
   https://dreampuf.github.io/GraphvizOnline/

2. Copy content from generated file

3. Paste into the editor

4. Diagram will be rendered visually

--------------------------------------------------
📦 DEPENDENCIES
--------------------------------------------------

Defined in requirements.txt:

- pandas
- networkx
- matplotlib
- pydot
- scikit-image
- openpyxl

--------------------------------------------------
🧠 NOTES
--------------------------------------------------

- Ensure Excel file format matches expected structure
- Only rows with "Yes / No" = "Yes" are processed
- Output files are text-based graph definitions (DOT format)
- These can be further converted to PNG/SVG if required
